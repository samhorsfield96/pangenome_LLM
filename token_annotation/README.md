## token_annotation

### annotate_tokens.py

Annotate tokenized gene clusters with functional information. Maps gene cluster representatives to annotation data, creating comprehensive token annotation files that link numerical tokens to gene names and functional descriptions.

**Usage:**
```
python annotate_tokens.py --clusters <cluster_file> --annotations <annotations.tsv> [--outpref <prefix>]
```

| Argument | Required | Default | Description |
|---|---|---|---|
| `--clusters` | Yes | — | Cluster file |
| `--annotations` | Yes | — | Path to gene annotation `.tsv` file |
| `--outpref` | No | `annotated_tokens` | Output prefix |

---

### annotation_to_token.py

Match tokenizer vocabulary with annotated token data. Filters annotation files to include only tokens present in the tokenizer vocabulary, ensuring compatibility between model inputs and functional annotations.

**Usage:**
```
python annotation_to_token.py --tokens <tokenizer.json> --annotations <annotations.tsv> [--outpref <prefix>]
```

| Argument | Required | Default | Description |
|---|---|---|---|
| `--tokens` | Yes | — | Path to tokenizer JSON file |
| `--annotations` | Yes | — | Path to `.tsv` file generated by `annotate_tokens.py` |
| `--outpref` | No | `annotated_tokens` | Output prefix |

---

### find_concurrent_genes.py

Identify genomes containing specific gene combinations. Searches for co-occurring genes within genomes, supporting single gene searches or pairwise gene combinations, and outputs filtered genome subsets for downstream analysis.

**Usage:**
```
python find_concurrent_genes.py --infile <tokenised_genomes.txt> --annotations1 <annotations1> [--annotations2 <annotations2>] [--outpref <prefix>]
```

| Argument | Required | Default | Description |
|---|---|---|---|
| `--infile` | Yes | — | Path to tokenised genome data `.txt` file |
| `--annotations1` | Yes | — | First path to annotated tokens from `annotate_tokens.py` or `calc_gene_frequency.py` |
| `--annotations2` | No | None | Second path to annotated tokens (for pairwise co-occurrence search) |
| `--outpref` | No | `comparisons` | Output prefix |

---

### get_gene_positions.py

Extract positional information for specific tokens across genomes. Maps token locations within genome sequences, supporting absolute ID conversion and generating position matrices for spatial analysis of gene distributions.

**Usage:**
```
python get_gene_positions.py --infile <tokenised_genomes.txt> --tokens <token1,token2,...> [--outpref <prefix>] [--absolute-IDs]
```

| Argument | Required | Default | Description |
|---|---|---|---|
| `--infile` | Yes | — | Path to tokenised genome data `.txt` file |
| `--tokens` | Yes | — | Comma-separated list of tokens to search for |
| `--outpref` | No | `plot` | Output prefix |
| `--absolute-IDs` | No | False | Combine forward and reverse strand genes using absolute gene IDs |

---

### token_likelihood_position.py

Analyze token likelihood distributions across genomic positions. Calculates position-specific token frequencies, generates statistical profiles, and creates visualizations of token positional patterns within genome assemblies.

**Usage:**
```
python token_likelihood_position.py --token-likelhoods <likelihoods_file> --genomes <genomes_file> [--topN <int>] [--profile-size <int>] [--ignore-breaks] [--ignore-others] [--outpref <prefix>]
```

| Argument | Required | Default | Description |
|---|---|---|---|
| `--token-likelhoods` | Yes | — | Token likelihoods generated from `compute_token_likelihood.py` |
| `--genomes` | Yes | — | Input genomes used for `compute_token_likelihood.py` |
| `--topN` | No | `5` | Number of elements to show in profile |
| `--profile-size` | No | `10` | Number of genes upstream and downstream to include in profile |
| `--ignore-breaks` | No | False | Ignore contig breaks in plotting |
| `--ignore-others` | No | False | Ignore genes outside of top-N frequency in regions |
| `--outpref` | No | `output` | Output prefix |
