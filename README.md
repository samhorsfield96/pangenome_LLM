# A repository of scripts for testing of PanBART, an LLM trained on pangenomes

## coselection_analysis

Scripts to aid in detection of signals of epistasis.

## embedding_visualisation

Scripts for interpreting and plotting embeddings.

## environments

Files for generating conda environments for analysis.

## misc

Miscellaneous analysis scripts.

## plotting_scripts

Scripts for generation of figures.

## QC

Scripts for quality control.

## sequence_generation

Scripts for simulating genomes from PanBART.

## token_annotation

Scripts for generating token-level annotations and analysing function of tokens.

## training_preparation

Scripts for preparing data for training of PanBART
