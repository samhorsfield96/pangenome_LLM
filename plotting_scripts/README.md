## plotting_scripts

### gene_freq_to_pseudolikelihood.R

Analyze relationship between gene frequency and pseudolikelihood scores. Merges gene frequency data with pseudolikelihood calculations, creates scatter plots with LOESS smoothing, and generates boxplots showing how gene frequency affects model predictions.

**Usage:**
```
Rscript gene_freq_to_pseudolikelihood.R --root <root_dir> --outdir <out_dir> --species <species1,species2> --species-names <"Name1,Name2">
```

| Argument | Required | Default | Description |
|---|---|---|---|
| `--root` | Yes | — | Root input directory containing one subdirectory per species |
| `--outdir` | Yes | — | Output directory for saved plots |
| `--species` | Yes | — | Comma-separated list of species subdirectory names (e.g. `E_coli,S_pneumoniae`) |
| `--species-names` | Yes | — | Comma-separated list of species display names (e.g. `"E. coli,S. pneumoniae"`) |
