## misc

### analyse_cosine.py

Analyze cosine similarity distances between tokens. Processes cosine similarity matrices, extracts upper triangle values above minimum thresholds, and annotates token pairs with functional annotations for downstream analysis of gene relationships.

**Usage:**
```
python analyse_cosine.py --infile <cosine_file> --annotations <annotations> [--outpref <prefix>] [--min-sim <float>]
```

| Argument | Required | Default | Description |
|---|---|---|---|
| `--infile` | Yes | — | Path to cosine output file generated by `compute_cosine.py` |
| `--annotations` | Yes | — | Path to `.tsv` file generated by `annotate_tokens.py` |
| `--outpref` | No | `cosine_distances` | Output prefix |
| `--min-sim` | No | `0.0` | Minimum cosine similarity to include in output |

### cluster_genomes.py

Cluster genomes based on Jaccard distance using kernel density estimation. Calculates pairwise Jaccard distances between genomes, identifies optimal clustering thresholds using KDE peak detection, and generates genome clusters using network connectivity analysis.

**Usage:**
```
python cluster_genomes.py
```

> **Note:** This script uses hardcoded input/output values. Edit the variables at the top of `main()` before running:
> - `infile` — path to tokenised genomes file (default: `tokenised_genomes.txt`)
> - `outpref` — output prefix (default: `synteny`)
> - `subsample` — number of pairs to subsample (default: `100000`)
> - `kde_bandwidth` — KDE bandwidth (default: `0.01`)
> - `kde_tolerance` — KDE tolerance (default: `0.2`)
